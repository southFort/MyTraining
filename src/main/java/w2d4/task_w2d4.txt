Необходимо создать программу, которая продемонстрирует утечку памяти в Java.
При этом объекты должны не только создаваться, но и периодически частично
удаляться, чтобы GC имел возможность очищать часть памяти.
Через некоторое время программа должна завершиться с ошибкой OutOfMemoryError
c пометкой Java Heap Space.

Задание со звездочкой: сделать чтобы ошибка OutOfMemoryError была в Permanent
Generation (или Metaspace)
