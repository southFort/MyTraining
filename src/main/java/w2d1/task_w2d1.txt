Необходимо разработать класс, реализующий следующий интерфейс:

void serialize (Object object, String file);

Object deSerialize(String file);

Методы выполняют сериализацию объекта Object в файл file и десериализацию
объекта из этого файла.

Обязательна сериализация и десериализация "плоских" объектов (все поля
объекта - примитивы, или String).

Задание "Под звездочкой": работать c любыми типами полей (полями могут быть
ссылочные типы).

Использовать готовые реализации (Jaxb, jackson и т.д.) нельзя!

Требования к заданию:

    Работоспособность кода
    Архитектурная корректность (ООП)
    Документация (JavaDoc)
    Следование Java Code Convention
    Корректная работа с ресурсами
    Предусмотреть возникновение исключительных ситуаций
